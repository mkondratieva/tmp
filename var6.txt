6.  В файле 1.txt  задан двумерный массив целых чисел с, возможно,  различным
количеством элементов в каждой строке. Элементы массива 
задаются через   пробелы. Строки двумерного массива разрывов не имеют,
а столбцы могут иметь разрывы. Ограничений на количество строк и количество 
чисел в строке  нет. Распечатать двумерный массив.

Найти количество максимальных элементов последнего столбца матрицы.

Для решения задачи не использовать массивы (списки). Для введения чисел из файла
использовать список чисел из строки файла без обращения к размеру списка и без 
использования других методов списка, за исключением перебора его элементов. 
Просматривать элементы этого списка можно 1 раз.

Ошибки (отсутствие файла, некорректные данные, пустой файл) обработать.  
Файл открывать один раз. Использовать только последовательный доступ.
